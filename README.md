# face_Recognition
<center><img width="1885" height="868" alt="Screenshot 2025-11-25 234727" src="https://github.com/user-attachments/assets/4edfa58d-cebf-4893-9e0b-a2fd7474fd43" /></center>

This web application demonstrates custom computer vision and visual search. It allows users to upload an image, associate a unique name or tag with it, and then instantly retrieve that tag by uploading the same (or a similar) image later.

## Key Features & Benefits

*   **Image Upload and Tagging:**  Users can upload images and associate them with custom names or tags.
*   **Visual Search:**  Upload an image, and the application will identify the associated tag, if found.
*   **Custom Computer Vision Demo:**  Showcases the potential of custom computer vision in a simplified web environment.
*   **Instant Tag Retrieval:** Provides nearly immediate retrieval of assigned tags based on image similarity.

## Prerequisites & Dependencies

*   **Web Browser:** A modern web browser (Chrome, Firefox, Safari, Edge) is required to run the application.
*   **JavaScript Enabled:** JavaScript must be enabled in the browser.
*   **Basic understanding of HTML, CSS, and JavaScript:** This is beneficial for understanding and modifying the code.

## Installation & Setup Instructions

1.  **Clone the Repository:**

    ```bash
    git clone https://github.com/PraveenWijewardana/face_Recognition.git
    cd face_Recognition
    ```

2.  **Open `index.html` in your browser:**

    Simply double-click the `index.html` file to open it in your web browser.

## Usage Examples

1.  **Add a Person:**

    *   In the "Add People" section, enter a name in the text field.
    *   Click the "Choose File" button to upload an image of the person.
    *   The image will be displayed as a preview.

2.  **Find a Person:**

    *   In the "Find People" section, click the "Choose File" button to upload an image.
    *   The application will attempt to identify the person in the image and display their name (if previously tagged).
